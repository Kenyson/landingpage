Dicas de Produtividade
Command Palette

Use Ctrl+Shift+P para acessar todos os comandos disponíveis rapidamente.

Quick Open

Ctrl+P abre arquivos rapidamente. Use @ para símbolos e # para linhas.

Multi-Cursor

Alt+Click para múltiplos cursores. Ctrl+D para selecionar próximas ocorrências.

Terminal Integrado

Ctrl+~ abre o terminal. Use para executar comandos sem sair do editor.

Git Integration

Veja mudanças, faça commits e resolva conflitos diretamente no editor.

Debugging

F9 para breakpoints, F5 para iniciar debug. Suporte completo a Node.js e browsers.
