Como Funciona o Sistema de Grid
Breakpoints Responsivos

O Bootstrap usa breakpoints para diferentes tamanhos de tela:

  • col- - Extra small (<576px)
  • col-sm- - Small (≥576px)
  • col-md- - Medium (≥768px)
  • col-lg- - Large (≥992px)
  • col-xl- - Extra large (≥1200px)
Sistema de 12 Colunas

Cada linha é dividida em até 12 colunas. As classes definem quantas colunas cada elemento ocupa.
